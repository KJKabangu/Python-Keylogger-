Python Keylogger (Educational Use Only)

This project demonstrates a simple keystroke‑logging script written in Python using the pynput library.
It is intended only for cybersecurity education, research, and authorized testing on machines you own.

Features

Logs keystrokes in real‑time

Works on Windows, macOS, and Linux

Uses the pynput library for keyboard event listening

Installation
pip install -r requirements.txt

Usage
python keylogger.py
